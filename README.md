# My Dotfiles

## Prerequisites

Ensure the following tools are installed:

- Wezterm
- Fish Shell as default Shell
- zellij
- neovim
- fzf
- zoxide
- fd
- starship
- stow

## Install

Install fisher for fish: <https://github.com/jorgebucaran/fisher>

Run `stow .` in the root of this repo then restart your terminal.
